

### Rules for good commit messages:

```
    Empty line between summary/heading and main text
    Short summary/heading less than 50 chars
    Start summary/heading with a capital letter
    No . at  the end of summary/heading
    Use the imperative imperative -> this commit will
    Main text should line break after 72 chars
    In main text ->  Code tells you how comments, tell you why
    
```
